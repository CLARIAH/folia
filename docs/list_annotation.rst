Description & Examples
-------------------------

The ``<list>`` element creates a list, a class and set may be associated with the list to indicate the type of list. As
always, this vocabulary is user-defined. Individual list items need to be wrapped in the ``<item>`` structure element, which in turn
allows paragraphs, sentences, words and other structural elements.
